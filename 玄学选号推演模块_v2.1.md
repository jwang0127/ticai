# 🧿 中西合参 · 玄学选号推演模块

> **版本**: v2.1  
> **适用模型**: GPT-4o / Claude 3.5 Sonnet / Gemini 2.0 Flash / DeepSeek-V3  
> **协议约束**: 所有输出必须附带 `disclaimer`，严禁暗示概率优势

方案B = 塔罗占星（Tarot + Astrology）

## 七、后端校验清单（Python伪代码）

\`\`\`python
def validate_scheme(lottery, scheme):
    # 1. method 必须是三个固定值之一
    assert scheme['method'] in ['西方数秘', '塔罗占星', '六壬', '梅花八卦', '奇门河洛']

    # 2. numbers 长度正确
    length_map = {'3d':3, 'p3':3, 'p5':1, 'qxc':1, 'ssq':1, 'dlt':1, 'kl8':1}
    assert len(scheme['numbers']) == length_map[lottery]

    # 3. 号码合法性
    for num_str in scheme['numbers']:
        if lottery in ['3d','p3']:
            assert 0 <= int(num_str) <= 999
        elif lottery == 'p5':
            assert 0 <= int(num_str) <= 99999
        elif lottery == 'qxc':
            assert len(num_str) == 7 and all(c.isdigit() for c in num_str)
        elif lottery == 'ssq':
            parts = num_str.split(' + ')
            reds = [int(x) for x in parts[0].split()]
            blue = int(parts[1])
            assert len(reds) == 6 and all(1<=r<=33 for r in reds)
            assert sorted(reds) == reds  # 升序
            assert 1 <= blue <= 16
        elif lottery == 'dlt':
            parts = num_str.split(' + ')
            fronts = [int(x) for x in parts[0].split()]
            backs = [int(x) for x in parts[1].split()]
            assert len(fronts)==5 and all(1<=f<=35 for f in fronts)
            assert sorted(fronts)==fronts
            assert len(backs)==2 and all(1<=b<=12 for b in backs)
        elif lottery == 'kl8':
            nums = [int(x) for x in num_str.split()]
            assert len(nums)==8 and all(1<=n<=80 for n in nums)

    return True
\`\`\`

## 八、注意事项

1. **干支推算**：建议在 Python 层预先算好 \`draw_date\` 的干支，作为 \`system_prompt\` 的上下文注入，减少 LLM 出错概率
2. **多彩种批量**：可一次传入 \`lotteries: ["ssq","dlt","kl8"]\`，让模型并行输出多个 JSON 对象
3. **免责声明**：前端展示时必须醒目显示 \`disclaimer\`，建议字号不小于正文
4. **防滥用**：后端添加频率限制，防止被用于高频自动购彩脚本

---

> **最后提醒**：这套模块输出的本质是 **结构化随机数生成器 + 文化叙事包装**。玄学取数路径保证了"每次输出都有据可查"，但不改变彩票的独立随机本质。把它当作一个有趣的交互实验即可。
