# 体彩号码实验室

面向 GitHub Pages 的超级大乐透、排列3、排列5、福彩3D、体彩7星彩、福彩双色球公开数据看板。页面展示最新已核验开奖、下一期开奖时间、每个玩法唯一综合推荐清单及相对模型评分。首页顶部提供各玩法导航按钮；复制文本仅包含玩法名称和候选数字。

首页保持简洁看板不变，另提供六个详情页：

- `/dlt/`：大乐透综合推荐5注、上期复盘与最近100期分析；
- `/pl3/`：排列3合并主榜、冷门保护和热门观察后推荐8注，组选3、组选6各保留5注；
- `/pl5/`：排列5合并后精简推荐6注、上期复盘与最近100期分析；
- `/fc3d/`：福彩3D合并后推荐8注，组选3、组选6各保留5注。
- `/qxc/`：体彩7星彩使用七位置束搜索独立模型推荐8注；
- `/ssq/`：福彩双色球使用红蓝分区、红球共现独立模型推荐5注。

排列3和排列5共享一套位置模型；福彩3D、体彩7星彩、福彩双色球以及超级大乐透各自使用独立模型。排列3、排列5和福彩3D均分别展示各位置分析，避免“数字出现但位置错误”被误当作直选命中。

排列3、排列5和福彩3D不再展示独立热门区、冷门区；生成器按固定配额合并为唯一清单，并在每期开奖更新时自动重新计算。

> 以上仅为公开信息整理后的娱乐分析，不构成任何购彩建议，请理性参考。模型相对评分不是真实中奖概率。

## 本地生成

```powershell
python -m pip install -r requirements.txt
python src/fetch_draws.py
python src/generate_dashboard.py
python -m http.server 8000 -d docs
```

访问 `http://localhost:8000`。官方接口若临时返回 403 或超时，抓取脚本会保留上一次已核验数据，并把状态写入页面数据。

## 自动发布

GitHub Actions 在每天北京时间22:10更新排列3、排列5和福彩3D；在每周日、周二、周五08:10更新大乐透，即在开奖前一天生成。亦可手动选择全部玩法、仅大乐透或每日数字玩法。仓库 Settings → Pages 的 Source 需选择 **GitHub Actions**。

## 数据源与日历

- 首选：中国体育彩票官方开奖接口与开奖信息页；
- 交叉核验：省级体彩网及公开历史页；
- 超级大乐透：周一、周三、周六 21:25 开奖（休市日除外）；
- 排列3、排列5：每日 21:25 开奖（休市日除外）。
- 福彩3D：每日 21:15 开奖（休市日除外），数据来自中国福利彩票官方接口；
- 体彩7星彩：每周二、五、日开奖；
- 福彩双色球：每周二、四、日开奖，数据来自中国福利彩票官方接口。

法定休市属于例外事件，应以中国体育彩票当期公告为准。生成器按常规日历计算，并在页面明确标注“休市日除外”。
